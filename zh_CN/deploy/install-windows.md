# Windows

本章节将指导您如何在 Windows 系统中通过 zip 的形式下载安装并启动 EMQX。

:::tip
在 Windows 系统中建议仅将 EMQX 用于开发测试，推荐使用 [Docker 安装](./install-docker.md)。
:::

1. 下载 [emqx-5.0.14-macos11-amd64.zip](https://www.emqx.com/downloads/broker/v5.0.14/emqx-5.0.14-macos11-amd64.zip)，解压。
2. 命令行下进入解压路径，启动 EMQX：

```shell
./bin/emqx start
```
